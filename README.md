# Comparison of Batch Normalisation vs Layer Normalisation vs Group Normalisation


## Normalisation

Normalization is a pre-processing technique used to standardize data. There are different kinds of normalisation:
* Batch Normalisation
* Layer Normalisation
* Group Normalisation
* Instance Normalisation
